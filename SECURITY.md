# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, email us at **opensource@musictechlab.io** with:

- A description of the vulnerability
- Steps to reproduce the issue
- Any potential impact or severity assessment

We will acknowledge your report within **48 hours** and work with you to understand and address the issue promptly.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |

## Disclosure Policy

- We will coordinate with you on disclosure timing
- We aim to release a fix within **30 days** of confirmation
- Credit will be given to reporters unless anonymity is requested

Thank you for helping keep this project and its users safe.
